# 2월 11일 협업일지

날짜: 2026-02-11  
이름: 안호성  
팀명: 3팀

## 오늘 맡은 역할 및 작업 내용

오늘은 중간 발표 준비를 위해 발표자료에서 TTS 파트와 SQL 데이터베이스 파트 작성을 마무리했다.  
기능 나열이 아니라 실제 구현 구조와 데이터 흐름이 드러나도록 정리하는 데 집중했다.

1. 발표자료 작성 (TTS / SQL)

- TTS 파트 작성 완료
  - ONNX 기반 음성 합성 구조 설명 정리
  - 입력 텍스트 → 전처리 → 모델 추론 → 오디오 반환 흐름 시각화
  - 워커 기반 비동기 합성 구조 요약
  - 자동 재생 이슈 원인 및 해결 로직 정리

- SQL 데이터베이스 파트 작성 완료
  - 대화 기록 저장 스키마 정리 (질문 / 답변 / 시간 / rating 구조)
  - 사용자 피드백 저장 테이블 구조 설명
  - 질문 -> 답변 -> 피드백 저장까지의 전체 데이터 흐름 단계화
  - 추후 데이터 분석 및 모델 개선을 위한 저장 목적 명시

2. 발표자료 구조 보강

- 설계 의도와 기술 선택 이유를 보강

## 오늘 작업 현황

- TTS 발표자료 작성 완료
- SQL 데이터베이스 구조 설명 슬라이드 작성 완료
- 중간발표용 자료 1차 완성

## 오늘 협업 중 제안하거나 피드백한 내용

- 다른 팀원 발표를 위해 TTS 및 데이터베이스 관련 내용 공유

## 오늘 분석/실험 중 얻은 인사이트나 발견한 문제점

- TTS는 모델과 파이프라인 구조 설명이 핵심

## 일정 지연이나 협업 중 어려웠던 점

- 전체 슬라이드 디자인 톤이 통일되지 않음
- 기술 설명을 줄이면서 깊이를 유지하는 균형이 어려웠음

## 오늘 발표 준비나 커뮤니케이션에서 기여한 부분

- TTS 합성 구조 및 비동기 처리 흐름 설명 자료 작성
- SQL 저장 스키마 및 피드백 기록 구조 정리

## 내일 목표 / 할 일
