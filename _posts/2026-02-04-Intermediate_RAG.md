# 2월 4일 협업일지

날짜: 2026-02-04  
이름: 안호성  
팀명: 3팀

## 오늘 맡은 역할 및 작업 내용

오늘은 파싱/청킹 고도화와 UI 구성에 집중했다.  
장우정 팀원의 코드를 바탕으로 파서와 청킹 방식을 개선하고,
웹 UI까지 연결했다.

1. 파싱 방식 확장 (장우정 팀원 코드 기반)

- **PDF 파싱을 2축으로 분리**해서 처리
  - **fitz(PyMuPDF)**: 페이지 렌더링/이미지 추출에 강점
  - **pdfplumber**: 텍스트/표 기반 추출 품질 보강
- **HWP는 olefile 기반 파싱**을 추가
  - 기존 단일 파서 대비 "실패 시 fallback" 루트를 확보
  - 문서 포맷별 파서 특성을 분리해 안정성 향상
- 핵심 변경 요지
  - "한 가지 파서로 모든 문서를 처리" -> "포맷별 최적 파서 조합"
  - 텍스트 누락/깨짐 발생 시 보조 경로로 재시도 가능

2. Web UI 구축

- **Gradio 기반 웹 UI** 연결
- 질의 -> 응답 -> 결과 표시 및 음성 재생 흐름을 UI에서 확인 가능하도록 구성
- 파싱/청킹 변경 결과를 UI에서 즉시 확인할 수 있는 상태로 정리

## 오늘 작업 현황

- 파서 체계가 확장되어 문서 범용성이 증가
- Gradio UI까지 연결되어 데모 가능 상태 확보

## 오늘 협업 중 제안하거나 피드백한 내용
