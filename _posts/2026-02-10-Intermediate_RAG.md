## 오늘 맡은 역할 및 작업 내용

오늘은 **오디오 플레이어 다변화**와  
**TTS/SQL 기능 관련 발표자료 제작**에 집중했다.

1. 오디오 플레이어에 mpv 추가

- 배경
    - ffplay 단일 의존은 환경마다 재생 품질/호환성 이슈가 있음
- 변경 내용
    - 기존 `ffplay` 외에 `mpv`를 선택 가능하도록 추가
    - 플레이어 선택 로직에서 `mpv` 분기 추가
- 기대 효과
    - 환경별 문제 회피 가능
    - 플레이어 선택 폭 확대로 운영 안정성 상승

2. TTS 및 SQL 기능 발표자료 제작 기여

- TTS 파트
    - ONNX 기반 합성 구조 설명
    - 워커 기반 비동기 합성 흐름 요약
    - 자동 재생 문제 해결 로직 정리
- SQL 파트
    - 대화 기록/피드백 저장 스키마 요약
    - 저장 흐름(질문->답변->rating) 단순화 포인트 강조
- 결과
    - 팀 발표자료에서 "실제로 구현된 기능"을 명확히 보여줌
    - 기술 구현의 의도와 구조가 전달 가능하도록 정리됨

## 오늘 작업 현황
